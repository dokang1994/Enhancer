# Memory Policy

AI memory is not trusted.

The Git repository is the durable memory.

At minimum, a new session must recover state from:

- `CONSTITUTION.md`
- `AGENTS.md`
- `PROJECT_STATE.md`
- `CURRENT_TASK.md`
- `SESSION_HANDOFF.md`

For architecture work, also read:

- `ARCHITECTURE.md`
- `DECISION_LOG.md`
