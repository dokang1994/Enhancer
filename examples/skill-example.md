# Skill Example

This is a conceptual example for the future Skill System.

```text
Skill: Repository Context Recovery
Trigger:
  session start
Required reading:
  CONSTITUTION.md
  AGENTS.md
  PROJECT_STATE.md
  CURRENT_TASK.md
  SESSION_HANDOFF.md
Steps:
  read documents
  summarize current state
  identify next task
  report risks
```

Key rule:

Skills are repository-stored instructions, not hidden model memory.
